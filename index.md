→ [[projects/|浏览项目]]

## 写作规则

| 你的状态 | 放这里 |
|---------|--------|
| 💭 脑子里只有模糊想法，一两句话 | `inbox/` |
| 📅 今天做了什么流水账 | `daily/` |
| 🎓 学完一个东西想整理 | `notes/learning/` |
| ✨ 有个想法想展开 | `notes/ideas/` |
| 🏗️ 开始一个新项目 | `projects/<项目名>/` |
| 📋 周末回顾 | `weekly/` |

> 定期回顾 `inbox/`，有价值的内容整理到 `notes/`，其余删掉。
