# 知识库使用指南

## CLI 命令速查

```bash
note capture "突然的想法"          # 快速捕获碎片 → inbox/
note daily                         # 今日日常记录 → daily/
note new learning "Rust 所有权"    # 新建学习笔记 → notes/learning/
note new idea "一个点子"           # 新建想法笔记 → notes/ideas/
note new project "项目名"          # 新建项目设计 → projects/<名>/
note weekly                        # 写周报 → weekly/
note timeline                      # 刷新时间线
note publish                       # 构建 + 推送 → 更新网站
```

## 模板说明

| 模板 | 用途 | status 流转 |
|------|------|-----------|
| `capture.md` | 碎片想法 | — |
| `daily.md` | 日常记录 | — |
| `learning-note.md` | 学习笔记 | seedling → budding → evergreen |
| `idea-note.md` | 想法成篇 | seed → sprout → evergreen |
| `project-design.md` | 项目设计 | ideation → designing → building → shipped → archived |
| `weekly-review.md` | 周报 | — |

## 网站

- 本地构建：`npx quartz build -o docs --concurrency 4`
- 本地预览：`npx quartz build -o docs --serve --port 8080`
- 发布：`note publish`
- 网站地址：`https://yuechuye.github.io/my-notes`（需在 GitHub Settings 开启 Pages）
